# Diffusion Backend Changelog

### Initial Release
- Stable Diffusion 1.5/2.1 support
- SDXL support
- Text-to-image generation
- Customizable inference steps

- Seed control for reproducibility

- Custom dimensions
- Batch generation
- Progress tracking
- Memory optimization
